# Video backgrounds

Place your background videos here with these exact names:

- `wall1.mp4`
- `wall2.mp4`
- `wall3.mp4`
- `wall4.mp4`
- `wall5.mp4`
- `wall6.mp4`
- `wall7.mp4`

Each section of the site switches to a different video as you scroll.
If a file is missing, the site still works with a purple gradient fallback.
